# MiniLens

MiniLens is an application to take screenshots and save them locally to the
machine, with additional options to upload the pictures to your own hosting
site.

# Features:

* Screenshot
    * Full
    * Partial
    * Selective window
* Minimises to tray
* Hotkey shortcuts
* Different file formats (bmp, jpeg, png)
